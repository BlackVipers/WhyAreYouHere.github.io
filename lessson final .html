First lets learrn about HTML.

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by languages such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

  • Developer of HTMl:  Tim Berners-Lee

  • When did he develop HTML: 1993

  • When did HTML became an official mark-up language : December 1999.

  • The current version of HTML: HTML 5

CSS

Cascading Style Sheets (CSS) is a style sheet language used for decorating or giving style to the document written in a markup language such as HTML. CSS is a language used Web-Development, alongside HTML and JavaScript

  • Developer of CSS:  Håkon Wium Lie (Lie was working with Tim Berners-Lee )

  • When did he develop CSS: October 10, 1994

  • The current version of CSS: CSS 3

Java Script

JavaScript often abbreviated as JS, JavaScript is high-level, often just-in-time compiled, It has curly-bracket syntax, dynamic typing, prototype-based object-orientation, and first-class functions. It is a zero-index programming language.Alongside HTML and CSS, JavaScript is one of the core technologies of the World Wide Web.

  • Developer of JS:  Brendan Eich

  • When did he develop JS: September 1995

  • The current version of CSS: Java Script ECMAScript 2018
